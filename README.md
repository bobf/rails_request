# RailsRequest

_Rails_ toolkit for debugging and inspecting requests.

## Installation

Add the gem to your `Gemfile`

```ruby
gem 'rails_request', '~> 0.1.2'
```

And rebuild your bundle:

```bash
$ bundle install
```
